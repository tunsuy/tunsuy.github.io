一、 UI Testing简介

1、UI Testing 是基于 XCTest 测试框架的。XCTest 作为 OCUnit 的替代者，目前是 iOS 单元测试框架不二之选，很多其他测试框架也基于 XCTest 封装。XCTest 有如下特点：

  • 测试用例需要继承 XCTestCase
  • 有类似 Junit 的 setup 或者 teardown方法
  • 还算不错的 Assertions
  • 和 Xcode 深度集成
  • 可以使用 Xcode server 的持续集成。支持 Swift 和 Objective-C
